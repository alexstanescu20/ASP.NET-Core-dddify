# 赞助

Dddify 是基于 MIT 许可的开源项目，文档长期免费开放。赞助完全自愿，不影响任何开发者访问、使用或参与贡献。

如果 Dddify 对你的学习、项目实践或团队建设有所帮助，欢迎通过赞助支持项目的持续维护。

## 赞助用途

赞助资金将主要用于：

- 核心代码开发与长期维护。
- Bug 修复、兼容性适配和版本发布。
- 文档、教程和最佳实践建设。
- 生产级实践方案验证与经验沉淀。
- 服务器和域名等基础设施费用。

## 个人赞助

个人用户可以通过微信赞助支持项目。赞助 ¥99 至 ¥199 任意金额，可永久加入 Dddify 会员服务微信群，获取框架使用、实践落地和问题排查相关答疑。

## 企业赞助

企业或组织可以通过赞助支持 Dddify 的持续发展。企业赞助商可优先反馈使用问题、实践诉求和改进建议，帮助项目更好地适配真实业务场景。

赞助权益如下：

- 官网首页展示赞助商信息。
- 文档内容页面展示赞助商信息。

赞助商展示内容应与开源社区、开发工具、技术服务或相关专业领域保持一致。具体展示形式以双方确认结果为准。
